== Description ==
Wumetax Moving Mode 是一款超輕量維護模式外掛。  
啟用後，管理員可於後台控制是否顯示維護頁面。  
訪客將看到 Apple 風格的簡約維護畫面，並提供管理員隱藏式登入入口。

== Features ==
* 後台可手動開關維護模式
* 強制不快取，確保即時生效
* Apple 風格簡約介面
* 管理員登入彈出視窗（可關閉 / 輸入檢查）
* 完全無額外資源載入，極度輕量

== Installation ==
1. 將外掛上傳至 `/wp-content/plugins/` 目錄
2. 在 WordPress 後台啟用外掛
3. 前往 **設定 → 維護模式** 控制維護頁開關
